# **App Name**: MuslimahSphere

## Core Features:

- Community Feed: Display posts and questions from community members in a chronological order.
- Prayer Timings: Display accurate prayer timings based on city search with API integration. Provides timings for Fajr, Dhuhr, Asr, Maghrib, Isha.
- Qur'an Explorer: Enable users to read, listen to, and search the Qur'an with translations. Implements a fallback API for Arabic text rendering.
- Resource Library: Provide access to articles, duas, and hadith searchable, paginated, and categorized for easy access.
- Open Forum Q&A: Enable community members to post questions and receive answers.  Uses an LLM as a tool that an admin may use or ignore in providing appropriate answers to user questions. Answers are added via Google sheets and synced to the platform.
- Newsletter Sign-Up: Collect user emails for newsletter subscriptions and stores the addresses using a Google Sheets API.
- Donation Integration: Integrate a secure PayPal popup for accepting donations.

## Style Guidelines:

- Primary color: Deep teal (#329F92) to represent growth and tranquility within the Islamic context.
- Background color: Soft beige (#F5F5DC) to provide a neutral, calming backdrop.
- Accent color: Muted gold (#BDB76B) to add an element of sophistication and elegance.
- Headline font: 'Playfair', serif, for headings to evoke a sophisticated and fashionable feel; body text: 'PT Sans', sans-serif, for body to be more readable.
- Use elegant, minimalist icons inspired by Islamic art patterns for navigation and interactive elements.
- Employ a clean, grid-based layout to ensure content is organized and accessible. Prioritize whitespace for readability.
- Incorporate subtle animations on hovers and transitions to enhance user engagement without being distracting.
